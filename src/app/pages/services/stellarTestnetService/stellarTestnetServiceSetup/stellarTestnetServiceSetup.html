Configuring Stellar Testnet Service

The Stellar Testnet Service will create these if they do not exist already:

  • TXLM currency
  • Group named user which is default & public
  • Account configuration named default which is default & primary
  • This setup will take you through configuring the service.
  • Please refer to the Stellar Testnet service documentation if you would like to integrate Stellar Testnet into your own front-end or service.
